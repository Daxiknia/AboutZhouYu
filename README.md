## 项目简介

本项目整理自微博瑜bot（美周郎顾曲阆苑）的1500凹瑜向文包投稿，经删除重复、增加新篇、去除超短篇、规范格式后得成。项目旨在收集各类以周瑜为主角或重要角色的同人文学作品，包括但不限于：

- 历史向同人
- 架空穿越
- 耽美同人

## 目录结构

```
.
├── [0]资料/           # 史料、考据等资料
├── [html格式]*/       # 古早时期36论坛网页格式的同人文收集
├── 作者：*/           # 有三篇及以上同作者的文章按此归类
├── [标签]文名 by 作者  # 一般格式
└── README.md          # 项目说明文件
```
