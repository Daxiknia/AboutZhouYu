## 项目简介

本项目整理自微博瑜bot（美周郎顾曲阆苑）的1500凹瑜向文包投稿，经删除重复、增加新篇、去除超短篇、规范格式后得成。项目旨在收集各类以周瑜为主角或重要角色的同人文学作品及孙吴的论文书籍资料，包括但不限于：

- 历史向同人
- 架空穿越
- 耽美同人

## 目录结构

```
.
├── [0]资料/           # 史料、考据等资料
├── [0]code/site/      # 文包资料检索页
├── [html格式]*/       # 古早时期36论坛网页格式的同人文收集
├── 作者：*/           # 有三篇及以上同作者的文章按此归类
├── [标签]文名 by 作者  # 一般格式
└── README.md          # 项目说明文件
```

## 检索页

项目内置了一个静态检索页：

```
[0]code/site/index.html
```

可直接用浏览器打开，也可通过 GitHub Pages 访问。检索页会读取 GitHub 仓库文件树，并按文件名、作者、标签、路径等信息生成可筛选的目录。

当前支持：

- 模糊查询标题、作者、标签和路径
- 多种筛选方式，包括标签、屏蔽标签、作者、类型与 `[0]资料` 模式
- 相关度、标题、作者、大小、路径排序
- 高频标签快捷筛选
- 结果标签按需折叠显示
- 结果区顶部固定显示条目统计与批量操作
- 作品合集识别与子文件展开查看
- 当前结果勾选与 ZIP 批量下载，压缩包内保留仓库目录结构
- 条目选择与统计按文件路径计算
- 鼠标悬停标题查看完整路径

如需从检索结果中排除某些路径，修改：

```
[0]code/site/ignore-paths.js
```

在 `window.SITE_IGNORE_PATHS` 数组中添加路径即可。写文件夹路径时，会忽略该文件夹下的所有内容。
